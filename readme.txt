== Description ==
= ChatBot for Telegram =

Create a Chat Bot for Telegram with the power of the WPBot. Supports Simple text Responses, conversational forms and more. 

ChatBot for Telegram is an Addon for <strong>WPBot Free</strong> version to create Telegram ChatBots. With this WPBot ChatBot addon you can utilize the installed WPBot on your WordPress site as a hub to respond to user questions and provide eMail, FAQ, Conversational Form support within the Telegram app.

> [Download WordPress ChatBot WPBot](https://wordpress.org/plugins/chatbot/)


This ChatBot addon <strong>extends</strong> the WPBot plugin's functionality and adds the ability for WPBot to chat with the users on Telegram. 

Optionally this Telegram ChatBot uses dialogflow API: https://cloud.google.com/dialogflow/

However, user can enable or disable Dialogflow API from Chatbot settings.

Terms of service: https://cloud.google.com/dialogflow/docs/terms-trial-edition

The plugin can work well without connecting to DialogFlow utilizing all of the WPBot free version's features.

> //-> Try our <strong>[FREE ChatBot for WordPress with AI - WPBot](https://wordpress.org/plugins/chatbot/)</strong>. It is an easy to use, Native, No coding required, AI ChatBot to provide Automated Live Chat Support. Use ChatBot to <strong>answer user questions</strong> and also <strong>collect information</strong> from the users using <strong>conversational forms</strong>. It can be also be powered by DialogFlow, Tavily or OpenAI ChatGPT or simply use the built-in features to provide Live support and collect user data. ++ Check out [ChatBot for WordPress](https://www.wpbot.pro/) Demo.

== Installation ==

1. Download the plugin zip file. Extract and upload in your wp-content/plugins folder.
2. From the wp-admin panel go to plugins and activate "Telegram addon"
4. You are done.


== Frequently Asked Questions ==
= How to set up Telegram Bot =
First of all you need to have WPBot Telegram Addon installed and activated on your website. Then you need to create a telegram bot. In order to create your own telegram bot you need to have a telegram account and telegram installed on your phone or computer.

There’s a bot for that. Just talk the BotFather and follow a few simple steps. Once you’ve created a bot you will receive your authorization token which we need to add in Bot – Telegram > Telegram Access Token field. That’s it. Now your telegram bot will use the WPBot to respond to user questions.
